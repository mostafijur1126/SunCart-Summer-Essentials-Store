# 🌞 SunCart – Summer Essentials Store

🔗 Live: https://sun-cart-summer-essentials-store.vercel.app/

## Overview

SunCart is a modern summer eCommerce platform where users can explore and purchase products like sunglasses, outfits, skincare, and beach accessories.

## Purpose

Provide a smooth and responsive shopping experience with authentication-based ordering.

## Features

- 🔐 Authentication (Better Auth)
- 🛍️ Product browsing
- 📄 Product details page
- 🛒 Order placement (auth required)
- 🎨 Modern UI & responsive design
- ⚡ Animations with Framer Motion

## Tech Stack

- Next.js
- React
- Better Auth
- Hero UI
- Tailwind CSS
- Framer Motion
- React Icons
- Gravity UI Icons
